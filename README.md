# export-vmmetadata
Powershell script to export vCenter metadata to an Excel file.

Right now, the script will export the following data for all VMs on multiple vCenter Servers.
* VM Creation date
* VM Hardware Version
* Attributes (multiple)
* Tags (multiple)
* Attached Disks (multiple)
* Network Adapters (multiple)
* Host information
*   Version
*   Build
* Memory Allocation
* Disk Allocation
* CPU Allocation
* VM Tools Version
* Power State
* Guest OS
* Snapshot Status
* Hosting Cluster
* Notes
